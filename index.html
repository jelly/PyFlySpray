Pyflyspray

Python Interface to Archlinux FlySpray bugtracker

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the PyFlySpray Page

With this tool you can easily access the Archlinux FlySpray Bugtracker without having to open your browser. This tool is mainly targeted for information gathering to be used in mail cronjobs or just on the command line. It supports the following features:

  • Fetching the unassigned bugs from the different trackers (Archlinux,Community,etc.).
  • Fetching open bugs since a given date.
  • Fetching open bugs by maintainer.
  • Find orphan packages bugs, this collects orphans from Archweb and fetches the open bugs from the Bugtracker
  • Fetch orphans from Archweb.

Requirements

  • python3
  • lxml
